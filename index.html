Cybersecurity Internship Report

1. Common Cybersecurity Threats

I learned about several common threats that organizations face, including:

  • Brute force attacks, where hackers try to guess passwords automatically, often targeting services like SSH, WordPress sites that have vulnerabilities, or admin dashboards.
  • Phishing, which involves emails (often impersonating a trusted person or an organization) or fake websites designed to steal login info or sneak in malware like Trojan horses or ransomware.
  • Malware: Ransomware, Trojans, or botnets.
  • C2 infrastructure, which hackers use to maintain remote control over compromised systems.

2. Red Team Mindset

Understanding cybersecurity defense means thinking like an attacker. The red team mindset involves performing careful reconnaissance (OSINT or aggressive scanning) to learn about your targets, spotting weaknesses and potential ways to exploit the target. It also involves using what we know about attacker tactics and behaviors to predict and replicate real attacks. Tools like the MITRE ATT&CK framework help map out and connect these attacker techniques.

3. MITRE ATT&CK Framework & Cyber Threat Intelligence (CTI)

The MITRE ATT&CK framework is a widely used knowledge base that maps out how adversaries operate. It helps security teams:

  • Understand attacker behaviors in a clear way.
  • Find areas where defenses may be weak.
  • Focus their efforts on the most important threats.

By using threat intelligence, teams can connect the dots between observed clues like malicious IP addresses, domains, or files, and known threat groups or campaigns. Cyber Threat Intelligence (CTI) gives extra context to raw data, turning it into useful insights that can guide better security decisions.

4. Hands-On CTI Use Case: Malicious IP Analysis

As a hands-on experiment, I analyzed the IP address 185.100.87.202, a known malicious Indicator of Compromise (IOC), using OSINT tools.

  • VirusTotal flagged the IP as malicious by 9+ vendors for phishing, malware, and command-and-control activity.
  • AbuseIPDB showed over 15,800 reports by 900+ sources that indicated brute-force SSH attack, port scan, and web application hacking.
  • Shodan revealed an open SSH port (22) with OpenSSH 9.2p1 Debian likely utilized for unauthorized remote use or C2.

Threat Infrastructure Context

This IP is hosted by FlokiNET Ltd, a Romanian hosting provider well known for privacy-focused hosting but typically abused by cybercriminals. The IP was detected in most threat reports over several years and remains an active threat.

MITRE ATT&CK Correlation - Mapped Tactics and Techniques:

  • SSH brute force attacks (T1110)
  • RCE (Remote Code Execution) via SSH (T1059)
  • Persistent access maintenance via server components exposed (open ports etc.) (T1505)
  • Use of SSH as a covert C2 channel (T1071)

Conclusion

The IP is a persistent malicious IP involved in brute-force, phishing, and malware operations. Its hosting environment and behavior align with tactics used by financially motivated cybercriminal groups. This exercise showed the importance of combining multiple intelligence sources and using frameworks like MITRE ATT&CK to build actionable threat profiles.
